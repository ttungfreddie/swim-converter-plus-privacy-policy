Swim Converter+ is a tool that allows you to convert swimming times between SCY/LCM/SCM with history and export capabilities.

Last updated: May 2026

Extension Version: 1.0.0

This privacy policy applies to the Swim Converter+ Chrome extension.

## Information We Collect
Swim Converter+ does not collect, transmit, or sell any personal information.

The only data stored (locally) by the extension is conversion history entered by the user (time, event, course type, and result)

## How Information Is Used
All stored data is used solely to:

- Display past conversions in the extension
- Allow users to delete individual entries
- Enable export of conversion history as a CSV file

## Data Storage and Retention
All data is stored locally on the user’s device using Chrome’s local storage API.

- Data is not uploaded anywhere
- Data is not shared with any third parties
- Data remains on the user’s device unless manually cleared by the user

## Data Sharing
Swim Converter+ does not share any data with external servers, services, or third parties.

## Permissions
NewsifAI requests only the minimum permissions required to operate, including:

The extension may request the following permissions:

- Storage: Used to save conversion history locally on the device
- Side Panel: Used to display the extension interface

These permissions are used strictly for functionality and not for tracking or collecting personal data.

## Data Deletion
Users can delete stored data at any time using the “Clear History” button inside the extension. Uninstalling the extension also removes all locally stored data.

## Changes to This Policy
This privacy policy may be updated in future versions of the extension. Any updates will be reflected in this document.

## Security
User data stays completely local on the user's device.
